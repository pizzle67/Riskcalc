## Docs
[Table Driven Tests](https://go.dev/wiki/TableDrivenTests)

## Setup
### 1. Write Unit Tests
Tests typically live adjacent to the files they're testing, for example `getting_started_test.go` is located next to `getting_started.go`.

### 2. Write Component Tests
Using the docs above, write table driven tests to test your application using mocks as appropriate.

### 3. Run Your Tests
Run `make test`
